# ⚽ Arbitro

**Sports management and analytics system for football referees.**

Arbitro is a Full-Stack web application that automates the retrieval of match assignment data (from the PZPN system) and integrates it with advanced health and performance analytics from Garmin watches. Forget about manual data entry – the system automatically links your heart rate and pace metrics to the correct match half!

🚀 **Project Status:** Local-First Version. 
You download and run the application exclusively on your local machine. This ensures your sensitive login credentials (PZPN, Garmin) are 100% secure and never leave your hard drive.

## 🛠️ Tech Stack
* **Backend:** Python 3, FastAPI, SQLite
* **Frontend:** Vue.js 3 (Composition API), Vite, Chart.js
* **Integrations:** Garmin Connect API, Web Scraping (PZPN Extranet)
* **Data Engineering:** Custom ETL pipelines, Time-Series Data Processing

## 📥 Local Installation & Setup

Follow the instructions below to get Arbitro up and running on your local machine.

### 1. Clone the repository
```bash
git clone [https://github.com/YOUR_USERNAME/Arbitro.git](https://github.com/YOUR_USERNAME/Arbitro.git)
cd Arbitro
```

### 2. Configure Environment
```bash
cd backend
touch .env
python3 -m venv venv
source venv/bin/activate
pip install requirements.txt
```
`GARMIN_EMAIL=your_email@example.com` \
`GARMIN_PASSWORD=your_garmin_password` \
`SURNAME_NAME="Surname Name"` \
`DB_PATH = 'arbitro.db'` \
`PZPN_EMAIL = yourpzpn_email@example.com` \
`PZPN_PASS = youe_pzpn_password` \
`CURRENT_SEASON = "2025/2026"`

### 3. Running the App
```bash
~/Arbitro/backend$ uvicorn main:app --reload
~/Arbitro/vue-arbitro$ npm install
~/Arbitro/vue-arbitro$ npm run dev
```
### 4. How to use
Open the local link generated by the Vue terminal in your browser, typically:
`http://localhost:5173`
